# వాస్తు శాస్త్ర సలహాదారు — PWA App

## 📱 Android Home Screen లో Install చేయడం (Free)

### Step 1 — GitHub Account
👉 https://github.com → Sign Up (free)

### Step 2 — New Repository
1. Click **"+"** → **"New repository"**
2. Name: `vastu-app`
3. Visibility: **Public**
4. Click **"Create repository"**

### Step 3 — Upload Files
Click **"uploading an existing file"** and upload ALL these files:
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

Click **"Commit changes"**

### Step 4 — Enable GitHub Pages
1. Go to repository **Settings**
2. Click **"Pages"** (left sidebar)
3. Source: **"Deploy from a branch"**
4. Branch: **main** → folder: **/ (root)**
5. Click **Save**

### Step 5 — Get Your URL
After 1-2 minutes your app is live at:
`https://YOUR-USERNAME.github.io/vastu-app/`

### Step 6 — Install on Android
1. Open the URL in **Chrome** on your Android phone
2. Tap the **3-dot menu (⋮)** at top right
3. Tap **"Add to Home screen"**
4. Tap **"Add"**
5. ✅ వాస్తు App మీ Home Screen లో కనిపిస్తుంది!
